# Portfolio - Aldo Jimenez Wiehoff

A Vim/Neovim-inspired portfolio website built with modern web technologies.

## Features

- **Vim-style Navigation**: Navigate using `hjkl` keys, command mode with `:`, and more
- **Dual Navigation**: Traditional mouse/click navigation also available
- **Responsive Design**: Works on all devices
- **Modern Stack**: Vue 3, TypeScript, Tailwind CSS, Pinia
- **Fast & Optimized**: Built with Vite for optimal performance

## Technologies

- **Frontend**: Vue 3 (Composition API)
- **Language**: TypeScript
- **Styling**: Tailwind CSS (Catppuccin-inspired theme)
- **State Management**: Pinia
- **Routing**: Vue Router
- **Build Tool**: Vite

## Installation
```bash
# Clone the repository
git clone https://github.com/AldoJimenezW/portfolio-vim.git
cd portfolio-vim

# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Vim Commands

- `:home` - Navigate to home page
- `:experience` - View work experience
- `:projects` - View projects
- `:blog` - Read blog posts
- `:about` - About me
- `:contact` - Contact page
- `:help` - Show help

## Keyboard Shortcuts

- `h/j/k/l` - Scroll left/down/up/right
- `?` - Show help
- `Esc` - Exit command mode
